---
title: User Management
---

1. To invite a new user or add an existing user to a Project that you
   administer, select the "Manage" button on the Project dropdown.
<img src="assets/dashboard/manage-users.png" style="display: block; margin: 0 auto;"/>
2. In the ensuing popup, you can manage the Project's details, invite new users
   and change permission levels of existing users.
<img src="assets/dashboard/manage-user-detail.png" style="display: block; margin: 0 auto;"/>
