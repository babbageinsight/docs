# Welcome to Babbage Insight's documentation!

## Authentication
Visit our Dashboard. Learn how to [sign up](auth/sign-up), [sign in](auth/sign-in) and [sign out](auth/sign-out).

## User Dashboard
If you're a new *User*, you should [join a Project](user/projects) or two. And optionally, manage your [personal Metrics](user/metrics).

Then, get to [My Insights](user/insights)!

## Administration
If you're an *Administrator*: 
* help [your users](admin/users) join a Project or two, and
* manage the [Data Sources](admin/sources), [Metrics](admin/metrics) and [LLMs](admin/llms) associated with the Projects you administer.
