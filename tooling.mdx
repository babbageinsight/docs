## Primary Tooling
| **Service**               | **Tool**                                                        | **Purpose**                                                                        | **Location**                                                                            |
| :------------------------ | :-------------------------------------------------------------- | :--------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------- |
| Analytics                 | [motherduck](https://www.motherduck.com/) (DuckDB in the cloud) | Analytical work (OLAP).                                                            | *us-east-1* (AWS, owner=motherduck)                                                     |
| Compute                   | Linux instance                                                  | CPU-intensive computations. Backend orchestration.                                 | *us-east-2* ([AWS Lightsail](https://aws.amazon.com/lightsail/), owner=Babbage Insight) |
| Database                  | PostgreSQL                                                      | Relational database to store generated insights for APIs and the frontend.         | *us-east-2* ([AWS Lightsail](https://aws.amazon.com/lightsail/), owner=Babbage Insight) |
| LLM                       | OpenAI                                                          | LM for prompting and fine-tuning. Currently, Babbage Insight only supports OpenAI. |     |
| User & Project Management | [Clerk](https://www.clerk.com/)                                 | Maintain users and projects.                                                       |     |
| Data Sources | [Fivetran](https://www.fivetran.com/) ("Powered by Fivetran" Connect Cards)  | Pulling data from the project source, for analysis. **User Credentials are NOT stored on Babbage Insight's servers.**                                                      |     |
